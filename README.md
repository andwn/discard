# Discard
Work in progress Android client for Discord.
Uses a modified [Discord4J](https://github.com/austinv11/Discord4J).

## Download
Coming soon(tm)

## Build
1. Clone the repo
2. Open the folder in Android Studio
3. Hit run
4. Pray

### Why make this?
- More freedom
- Less memes
- None of my friends use Mumble
